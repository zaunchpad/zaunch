## What's Covered

This demo walks you through:

### Creating a Token Launch
- Setting up token information and branding
- Configuring sale parameters
- Deploying your token launch on the blockchain

### Participating in a Sale
- Browsing available token launches
- Making deposits using NEAR Intents
- Understanding the privacy features

### Claiming Tokens
- Loading your ZK proof
- Submitting claim transactions
- Receiving tokens in your wallet

## Key Features Demonstrated

- **Multi-Chain Support**: See how Zaunchpad works across Solana and NEAR
- **Privacy Features**: Understand how Zcash shielded pools protect your identity
- **NEAR Intents**: Watch seamless cross-chain deposits in action
- **ZK Proof System**: See how zero-knowledge proofs enable anonymous participation
